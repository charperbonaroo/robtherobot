## MessageListItem

Messages can be used to render a list of messages. `<MessageListItem />` elements represent the actual messages.

A user's message has a light background and padding...

<StoryCanvas>
  <MessageListItem
    message={{role: "user", content: "Hello"}} />
</StoryCanvas>

While an assistant's message is completely borderless and has no background.

<StoryCanvas>
  <MessageListItem
    message={{role: "assistant", content: "Hello, how can I help you?"}} />
</StoryCanvas>

## MessageList

A `<MessageList />` renders an array of messages

<StoryCanvas>
  <MessageList messages={[
    {role: "user", content: "Hello"},
    {role: "assistant", content: "Hello, how can I help you?"}
  ]} />
</StoryCanvas>

## MessageForm

A `<MessageForm />` renders a form that submits a message and clears its content.

<StoryCanvas>
  <MessageForm onMessage={action("onMessage")} />
</StoryCanvas>

You can disable the submit action using `<MessageForm disabled />`. This is useful if you want to allow the user to type a next message, but you don't allow a new message (yet).

<StoryCanvas>
  <MessageForm onMessage={action("onMessage")} disabled />
</StoryCanvas>
